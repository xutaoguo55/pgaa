Description:
PGAA provides Python and R implementations of distribution-aware single-cell perturbation statistics, including a Wasserstein statistic for full expression-distribution shifts and a persistent-homology statistic for responder-associated expression-shape changes. This archive is a code-only software release and excludes manuscript drafts, submission materials, figures, source-data tables, and review materials.
